# WorkTree-iD

A collaborative project planning platform where teams can organize tasks, add notes, and track completion in real-time.

## 🚀 Live Demo
[https://worktree-id.web.app/](https://worktree-id.web.app/)

## ✨ Features
- Real-time collaborative task management
- Add notes and links to tasks
- Mark tasks as complete
- Firebase integration for live sync

## 🛠️ Tech Stack
- React + TypeScript
- Firebase (Firestore, Authentication, Hosting)
- Vite

## 🏗️ Status
Currently in active development. Building iteratively with regular commits.
